# Design Audit

- Visual direction: restrained blue-gray academic identity on a white surface system.
- Layout width: 1160px maximum; reading width: 74ch.
- Typography: system font stack only; no downloaded font assets.
- Page structure: responsive Grid and Flexbox; no float layout.
- Components: content-led cards without fixed equal heights.
- Navigation: desktop inline navigation and lightweight accessible mobile menu.
- Interaction: keyboard focus treatment, Escape-to-close mobile navigation, and reduced motion by design.
- Bilingual behavior: every language switch maps to the same content route.
- Print: A4-oriented CV stylesheet removes navigation and decorative surfaces.
- Avoided: commercial landing-page patterns, glass effects, decorative gradients, large animation, and legacy blog chrome.
